## Project Overview

This repository is a focused Python practice lab designed to sharpen core programming fundamentals through compact scripts.

It currently includes:

- Pattern-printing challenges using nested loops
- Series-based math practice with factorial
- Clean, standalone scripts for quick execution and revision

If your goal is to build strong logic from the ground up, this repo is built for exactly that.

## Pattern Collection (Nested Loops)

The `Nested loop` folder contains nine progressive pattern scripts.

What this section trains:

- Loop nesting fluency
- Row-column iteration logic
- Space and symbol alignment techniques
- Confidence for common coding interview pattern rounds

Suggested learning flow:

1. Run each file and observe output carefully.
2. Rewrite each pattern from memory.
3. Modify dimensions or symbols to test your understanding.

## Series Module

The `Series/factorial.py` script focuses on factorial logic, a foundational concept used in:

- Combinatorics and permutations
- Recursion and iterative design comparisons
- Mathematical problem-solving patterns

## Getting Started

### Requirements

- Python 3.x

### Run Any Script

From the project root:

```powershell
python "Nested loop/pattern-1.py"
python "Nested loop/pattern-2.py"
python "Series/factorial.py"
```

If `python` is not available in your terminal, try:

```powershell
py "Nested loop/pattern-1.py"
```
